개인적으로 관심가졌던 책인데, 좋은 기회가 생겨 [DevOps 스터디](/thoughts/공부/devops-study/)로 같이 읽었다.

아직 경력이 없기 떄문에 IT 엔지니어에게 얼마나 도운되는 내용인지 감히 말할 순 없지만, 개인적으론 내용이 너무 만족스럽고 풍족하다고 느꼈다. 개발하거나 인프라를 건들 때 꼭 알아야 하는 개념을 적절히 설명해주는 책인 것 같다. 너무 이론적으로 가지 않으면서도, 충분한 깊이의 설명이 담겨있어서 좋았다.

네트워크 개념들을 기초적으로 공부하고자 하는 사람이 있으면 강력하게 추천할 수 있는 책이다!
