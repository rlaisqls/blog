## 1. 전기 (Electricity)

### 1-1. 전기와 전기현상

#### 전기의 발견과 발전

| 연대 | 주요 업적(인명, 국가) | 주요 내용 |
|------|-----------------|--------|
| BC 600년 | 마찰 전기 발견 (탈레스, 그리스) | 호박(electron)을 마찰하면 대전하여 물체를 흡인하는 것을 발견 |
| 1800년 | 전지의 발명 (볼타, 이탈리아) | 묽은 황산(전해액)에 구리판(+)과 아연판(-)을 세우면 약 1.1V의 기전력이 생김 |
| 1827년 | 옴의 법칙 (옴, 독일) | 전류의 세기와 전선의 길이에 반비례 관계에 있음과 전선의 저항을 밝혀냄 |
| 1831년 | 전자기 유도 (패러데이, 영국) | 코일 근처로 자석을 움직였을 때 전류가 생기는 것을 발견 (발전기) |
| 1840년 | 줄의 법칙 (줄, 영국) | 도체에 전류가 흐를 때 생성되는 열에너지에 대한 물리법칙 |
| 1876년 | 전화 발명 (벨, 미국) | 음성을 전기신호로 바꾸어 먼 곳에 전송하고, 이 신호를 다시 음성으로 재생 |
| 1879년 | 백열 전구 발명 (에디슨, 미국) | 높은 온도를 이용해서 빛을 내는 전기적인 조명 장치(필라멘트: 텅스텐) |
| 1885년 | 전자파 발견 (헤르츠, 독일) | 전자장의 진동이 파동의 형태로 전파되는 현상을 맥스웰이 예언하고 헤르츠가 실증 |
| 1888년 | 발전기 발명 (지멘스, 독일) | 역학적 에너지를 전기 에너지로 변환하는 장치로 일반적으로 전자기 유도를 이용 |
| 1890년 | 전자 발견 (톰슨, 미국) | 진공방전관을 실험하다가 0.1Pa의 낮은 기압에서의 진공방전에서 얇은 연두색의 형광빛 발견 |
| 1895년 | 무선 전신기 발명 (마르코니, 이탈리아) | 전압을 높이는 유도 코일, 불꽃 방전기, 수신기가 달린 코히러(무선 신호 검출기)가 부착된 기구로 실험 |

#### 원자 모델

- **Rutherford-Bohr model** (1913)
- 물질 → 분자 → 원자 → 양성자, 중성자, 전자
- **원자핵**: 고정되어 있음, 양전하(양전기)
- **전자**: 원자핵 주위를 돌고 있음, 음전하(음전기)
- 전자 1개의 전하량: **e = 1.6 x 10⁻¹⁹ [C]**
- 1C의 전하에는 6.24 x 10¹⁸개의 전자가 존재
- 원자에는 양전하와 음전하의 양이 같음 = 전기적으로 중성

#### 전하간에 작용하는 힘

- **쿨롱의 법칙**: F = (1/4πε) x (Q₁Q₂/r²)
- **전류**: i = dq/dt

#### 전기장(전계)과 전압

- **전기장**: E = F/Q₂ = (1/4πε) x (Q₁/r²)
- **전위차(전압)**: V = -∫(B→A) E·dL
- **전위**: V = -∫(∞→A) E·dL

### 1-2. 물질의 전기적 특성

#### 물질의 종류

- **구리(Cu, 원자번호 29)**: 금속 결합 → 전기가 잘 흐름
- **실리콘(Si, 원자번호 14)**: 공유 결합 → 전기가 잘 안 흐름

#### 도체와 유전체

- **도체**: 금속양이온과 자유전자로 구성, 전기가 잘 흐름
- **유전체**: 쌍극자로 구성, 전기가 잘 흐르지 않음

## 2. 자기 (Magnetism)

### 2-1. 자기와 자기현상

#### 자기(magnetism)

- 자기장(magnetic field)에 의해 생성되는 물리 현상
- 자기장(자계)의 생성요소: **전기 전류**, **자석(magnet)**

#### 자기력

- 자기에 의해 발생하는 힘
- 자기력선: N극에서 나와 S극으로 들어감
- 같은 극끼리: 척력(반발력), 다른 극끼리: 인력

#### 자극(magnetic pole)

- 자석에 있는 두 개의 극(N, S) → 정전기의 양전하와 음전하와 유사
- 자석은 언제나 두 개의 자극을 동시에 가지고 있음
- 자극의 세기(자하량)의 단위: **웨버[Wb]**

#### 자기에 관한 쿨롱의 법칙

F = k(m₁m₂/r²) = (1/4πμ₀)(m₁m₂/r²) [N]

- μ₀ = 4π x 10⁻⁷ [H/m] : 진공 중의 투자율(magnetic permeability)
- μᵣ : 비투자율(relative magnetic permeability)
- μ = μ₀μᵣ [H/m]

#### 자기장(자계)과 자속

- **자기장(자계)**: H = F/m = (1/4πμ₀) x (m₁/r²) [A/m]
- **자속**: Φ = m [Wb] (자극의 세기)
- **자속밀도**: B = μ₀H [Wb/m²] or [T]

### 2-2. 자기유도와 자성체

#### 자기유도

| 용어 | 설명 |
|------|------|
| **자기유도**(magnetic induction) | 자석 근처로 물체를 가져가면 물체가 자석이 되는 현상 |
| **자화**(magnetization) | 물체가 자성을 띠는 것 |
| **자성체**(magnetic substance) | 자기유도에 의해 자성을 띠는 물질 |

#### 자성체의 종류

- **강자성체**(ferromagnetic substance): 자기유도에 의해 강한 자성을 띠는 물질. 자석이 없어도 자화된 상태가 유지됨(잔류자기 → 영구자석)
- **상자성체**(paramagnetic substance): 외부 자석의 극과 반대 극으로 자화되는 물질 (철, 크롬 등)
- **반자성체**(diamagnetic substance): 외부 자석의 극과 같은 극으로 자화되는 물질 (구리, 납, 아연 등)

#### 자성의 원인

- 원자 내 전자의 궤도 운동과 스핀이 자기장을 형성
- 하나의 원자를 자석으로 생각할 수 있음
- 강자성체의 특성: 전자 스핀이 집단적으로 정렬됨, 자기장이 없어도 자성을 유지

### 2-3. 전류와 전자유도

#### 전류의 자기작용

- 직선 도선의 자기장의 세기: **H = I/(2πr) [A/m]**
- 솔레노이드(코일)의 자기장의 세기: **H = NI/d [A/m]**

#### 전자력

- **플레밍의 왼손 법칙**: 전자력의 방향을 결정
- **F = BIl [N]**

#### 전자유도

- **플레밍의 오른손 법칙**: 유도 기전력의 방향을 결정
- **e = Blv [V]**

## 3. 옴의 법칙

### 3-1. 전압과 전류

#### 전자의 전하량(전기량) = 1.6 x 10⁻¹⁹C

- 서로 다른 극성을 가지는 전하 간의 힘 → **인력**
- 서로 같은 극성을 가지는 전하 간의 힘 → **척력**(반발력)
- 6.25 x 10¹⁸개 전자들의 전하량의 합이 1C

#### 쿨롱의 법칙

F = k(Q₁Q₂/r²), k = 9 × 10⁹

- 전기력은 전하량에 비례하고, 거리의 제곱에 반비례

#### 전위(potential)

- 전하가 다른 전하에 인력이나 척력을 작용시켜 다른 전하를 이동시킬 수 있는 능력
- **전압 = 전위차** = 두 개의 다른 전하량을 가지는 전하에 의한 전위의 차이

#### 전압의 단위: 볼트(V)

v = W/Q [V] 또는 [J/C]

- 전위차 1V: 1C의 전하를 이동시키는데 1J의 에너지가 소비되는 경우의 전압

#### 전류(current)

- 두 전하 사이에 형성된 전위차에 의한 전하의 이동 흐름
- 전류의 단위: **[A]** (1A: 1초 동안에 1C의 전하가 어떤 한 단면을 지날 때의 전류량)

i = Q/t [A] 또는 [C/sec], i = dQ/dt

#### 전류의 방향

- 전류의 방향은 화살표로 표시
- 실제 전류가 흐르는 방향을 나타내는 것은 아님 (부호도 함께 고려)
- **전자의 이동 방향과 반대 방향이 전류의 방향**

#### 전류의 종류

| 종류 | 설명 |
|------|------|
| **직류(DC)** | 전류의 값이 시간이 지나도 변하지 않고 일정한 상수값을 가지는 전류. 대문자 I로 표기 (예: 건전지) |
| **교류(AC)** | 시간에 따라 값과 방향이 변하는 전류. 소문자 i로 표기 (예: 220V 가정용 전원) |
| **맥류** | 교류전원을 정류하여 만든 직류전원. 다이오드 브리지 회로를 이용 |

#### 직류값과 교류값

- **직류값 = 평균값**: (1/T)∫₀ᵀ f(t)dt (정현파의 경우: 0)
- **교류값 = rms값 = 실효값**: √((1/T)∫₀ᵀ f²(t)dt) (정현파의 경우: (1/√2)K = 0.707K)
- ω = 2πf = 2π/T (ω: 각속도, f: 주파수, T: 주기)

#### 수동부호규약

- 전류에 대한 기준 방향과 전압에 대한 기준 극성은 전적으로 임의로 지정 가능
- 일단 기준을 지정하면 선택된 기준과 일치하도록 모든 회로방정식을 표현해야 함
- **수동부호규약의 만족**: 전류의 방향이 회로소자에서의 전압강하를 발생시키는 방향인 경우 → 수동소자

### 3-2. 전원과 전력

#### 옴의 법칙

v = iR

- **저항(R)**: 전류의 흐름을 방해하는 정도. 단위: 옴[Ω]
- **컨덕턴스(G)**: 저항의 역수 G = 1/R. 단위: 지멘스[S]

#### 전력과 에너지

- **전력**: p = vi [W]
- **에너지**: w = ∫p(t)dt [J]

## 4. 키르히호프의 법칙

### 4-1. 회로 용어

| 명칭 | 설명 |
|------|------|
| **마디(node)** | 2개 이상의 회로소자가 연결되는 한 점 |
| **경로(path)** | 각 마디가 단 한 번만 지나가도록 여러 마디와 소자로 구성되는 통로 |
| **루프, 폐경로** | 출발 마디와 끝나는 마디가 같은 경로 |
| **가지** | 하나의 소자와 그 소자의 양쪽 끝에 있는 마디로 구성되는 단일 경로 |
| **메쉬(mesh)** | 루프 중에서 내부에 다른 루프를 포함하지 않는 루프 |
| **평면회로** | 어떠한 가지도 다른 가지의 위나 아래로 겹치지 않는 회로 |

#### 루프전류와 마디전압

- **루프전류**: 하나의 루프에 흐르는 공통된 전류 → **메쉬해석법**
- **마디전압**: 각 마디에 정의되는 하나의 전압 → **마디해석법**

### 4-2. 키르히호프의 법칙

#### 키르히호프의 전류 법칙 (KCL)

- 회로에서 마디는 회로소자가 아니므로 전하가 저장, 파괴, 생성되지 않음
- → **전하보존법칙** → 마디에서 전류의 합은 0 → **ΣI = 0**
- 마디로 유입되는 전류의 합 = 마디로 유출되는 전류의 합

#### 키르히호프의 전압 법칙 (KVL)

- 회로소자에서 소비하는 에너지의 합과 공급하는 에너지의 합이 같음
- → **에너지보존의 법칙** → 폐경로를 일주할 때 전압의 합은 0 → **ΣV = 0**
- 폐경로에서 전압강하의 합 = 전압상승의 합

### 4-3. 전류와 전압의 분배

#### 저항의 직렬 연결과 전압 분배

- 저항을 직렬로 연결하면 총 저항의 크기는 증가
- 전압 분배: 각 저항에 걸리는 전압은 저항 비에 비례

#### 저항의 병렬 연결과 전류 분배

- 저항을 병렬로 연결하면 총 저항의 크기는 감소
- 전류 분배: 각 저항에 흐르는 전류는 저항 비에 반비례

## 5. 마디해석법과 메쉬해석법

### 5-1. 마디해석법

#### 기준마디(reference node)

- 회로에서 전압을 정의하기 위해 기준으로 정하는 마디
- 기준마디의 선정: 가장 많은 가지를 갖는 마디를 선정 → 대지접지나 섀시접지를 선정

#### 마디전압(node voltage)

- 기준마디를 기준으로 마디의 전압을 지정하는 것
- 마디전압의 값을 결정하면 각 소자의 전압, 전류 관계를 알 수 있게 됨

#### 마디해석법의 과정

1. 기준마디를 선정하고 각 마디에 마디전압을 지정
2. 회로 내의 각 마디에 대한(기준마디 제외) 키르히호프의 전류 법칙(KCL)을 이용하여 회로방정식을 유도
3. 회로 방정식을 풀어 마디전압을 구함

#### 크라머(Cramer) 공식

- 행렬식 연산을 통해 연립방정식의 해를 구하는 방법
- x₁ = D₁/D, x₂ = D₂/D, x₃ = D₃/D

#### 종속전원과 마디해석법

- 독립전원이 포함된 회로에 대한 마디해석법과 동일한 과정으로 진행
- 종속전류원의 변수를 마디전압의 함수로 표현하여 방정식을 추가

#### 수퍼마디와 마디해석법

- 전압원이 두 마디 사이에 연결된 경우, 두 마디를 포함하는 수퍼마디를 구성
- 수퍼마디에 대해 KCL을 적용

### 5-2. 메쉬해석법

#### 메쉬해석법의 과정

1. 메쉬전류를 선정
2. 키르히호프의 전압 법칙(KVL)을 적용
3. 연립방정식의 해를 구함

#### 수퍼메쉬해석법

- 전류원이 두 메쉬에 공유되는 경우, 수퍼메쉬를 구성

#### 마디해석법과 메쉬해석법의 비교

| 마디해석법 | 메쉬해석법 |
|---------|---------|
| 마디전압 선정 | 메쉬전류 선정 |
| KCL(전류 법칙) 적용 | KVL(전압 법칙) 적용 |
| 전류원이 많은 회로에 유리 | 전압원이 많은 회로에 유리 |

## 6. 커패시터와 인덕터

### 6-1. 커패시터

#### 커패시터(capacitor)의 기본 구조

- 금속인 2개의 평행판으로 이루어지며, 두 평행판은 유전체(dielectric)로 불리는 절연물질에 의해 격리되어 있음
- 전기에너지를 전기적으로 저장함

#### 커패시터의 전하 저장 과정

1. 중성상태 → 전자의 이동 진행 → 전자의 이동 중지(vAB = vS) → 저장된 전하의 유지

#### 커패시턴스

- 커패시터가 평행판 양단에 저장할 수 있는 전하량을 단위전압으로 환산한 값
- 단위: **패럿(F)**

C = Q/v [F], i = C(dv/dt)

#### 커패시턴스와 커패시터의 구조

C = ε(A/d)

- ε: 유전율(permittivity)
- 커패시턴스는 평행판의 면적이 클수록, 평행판 사이의 간격이 작을수록 커짐

#### 커패시터의 단자 특성

v(t) = (1/C)∫₋∞ᵗ i(τ)dτ = v(t₀) + (1/C)∫ₜ₀ᵗ i(τ)dτ

- v(t₀): 커패시터의 초기전압

#### 커패시터의 정격전압

- 커패시터에 손상을 주지 않고 인가할 수 있는 최대의 직류전압

### 6-2. 커패시터의 에너지 저장

#### 커패시터 저장 에너지

W_C(t) = ½Cv²(t)

#### 커패시터의 충전과 방전

- **충전(charge)**: 전하가 커패시터의 평행판에 저장되어 에너지를 전기장 내에 저장하는 과정
- **방전(discharge)**: 충전된 커패시터에 저항이 연결되면, 커패시터에 충전된 전하가 흘러나와 저장된 에너지를 저항에 공급하는 과정

### 6-3. 커패시터의 연결

#### 커패시터의 직렬 연결

1/C_eq = 1/C₁ + 1/C₂ + 1/C₃

- 직렬 연결된 커패시터의 등가 커패시턴스는 각 커패시턴스의 역수를 합하여 다시 역수를 취하면 구할 수 있음

#### 커패시터의 병렬 연결

C_eq = C₁ + C₂ + C₃

### 6-4. 인덕터

#### 인덕터의 구조

- 도선을 코일 모양으로 된 구조
- 전기에너지를 자기적으로 저장함

v_L = L(di_L/dt)

#### 인덕터의 특성

- **초기**: 개방(open) 상태처럼 동작 (초개말단)
- **정상상태**: 단락(short) 상태처럼 동작

#### 인덕터에 저장되는 에너지

W_L(t) = ½Li²(t)

#### 인덕터의 직렬 연결

L_eq = L₁ + L₂ + L₃

#### 인덕터의 병렬 연결

1/L_eq = 1/L₁ + 1/L₂ + 1/L₃

### 6-5. 커패시터와 인덕터의 연속성 조건

- **커패시터의 연속성 조건**: 커패시터의 전압은 급격하게 변할 수 없음 (전압의 연속성)
- **인덕터의 연속성 조건**: 인덕터의 전류는 급격하게 변할 수 없음 (전류의 연속성)

#### 커패시터와 인덕터의 특성 비교

| 특성 | 커패시터 | 인덕터 |
|------|---------|---------|
| 에너지 저장 방식 | 전기장(전기적) | 자기장(자기적) |
| 전압-전류 관계 | i = C(dv/dt) | v = L(di/dt) |
| 저장 에너지 | W = ½Cv² | W = ½Li² |
| 직류 정상상태 | 개방(open) - 초단말개 | 단락(short) - 초개말단 |
| 직렬 연결 | 역수의 합의 역수 | 합 |
| 병렬 연결 | 합 | 역수의 합의 역수 |
| 연속성 조건 | 전압이 급변 불가 | 전류가 급변 불가 |

## 부록: 정보통신 개요

### 정보기술(IT, Information Technology)

- **신호**: 데이터가 전기적 형태로 표현된 것
- **데이터**: 처리 대상, 자료, 관측치
- **정보**: 가치 있는 데이터, 처리된 데이터
- **정보통신기술**: ICT (Information Communication Technology)

### 정보통신망(ICT Network)

- **통신 프로토콜**: TCP/IP
- **단말기**: 전화기, 컴퓨터, 휴대폰 (웹 클라이언트)
- **통신 매체**: 꼬임선, 동축선, 광케이블
- **통신 장비**: 교환기, 허브, 라우터, 게이트웨이
- **근거리통신망(LAN)** / **광역통신망(WAN)**

### 전기회로와 전자회로

- **전기회로**: 전원(전기에너지공급, 전압생성) + 저항(전기에너지소모: 열, 빛), v = iR (옴의 법칙)
- **전자회로**: 반도체 소자 사용
  - 다이오드 → 정류
  - 트랜지스터 → 증폭, 스위치
